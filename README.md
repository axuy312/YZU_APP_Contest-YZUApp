## YZU Pass App Feature
* 我們致力於讓學生可以**方便且自由的進出入學校校館**，不再讓學生受到學生證遺失之苦。
* 我們希望能讓學校邁入數位化時代，**增進管理效率和減少印製成本**。
* 增加安全管理功能，隨時可以**查看自己的刷卡記錄**。
